# DAY4-
作业1
本题使用 Titanic 数据集完成了数据清洗与预处理。首先读取数据并查看缺失值情况，然后分别使用删除、填充和插值三种方法处理缺失值。对于缺失较严重的 Cabin 字段，选择删除；对于 Age 字段，使用中位数填充，并演示了插值法；对于 Embarked 字段，使用众数填充。之后检查并删除重复记录，使用 IQR 方法识别并处理 Fare 字段中的异常值。最后对类别变量进行了类型转换，并统一字段格式，保存清洗后的数据。通过本题，我理解到数据分析前必须先保证数据质量，否则后续统计和建模结果可能不可靠。
作业2
本题使用北京 PM2.5 空气质量数据进行分析。首先读取数据并构造 datetime 时间字段，然后对 PM2.5 缺失值进行插值处理。接着计算 PM2.5 的均值、中位数、最大值、最小值和标准差，了解污染物的整体水平和波动情况。在可视化部分，使用折线图展示 PM2.5 随时间变化的趋势，使用柱状图比较不同月份和季节的 PM2.5 平均水平，使用散点图观察温度与 PM2.5 的关系，并使用热力图展示多个数值变量之间的相关性。通过分析可以初步判断，空气质量数据具有明显的时间变化特征，部分月份或季节的污染水平可能更高。
